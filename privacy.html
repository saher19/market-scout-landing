Политика конфиденциальности

Дата вступления в силу: 21 апреля 2026 г.

Приложение Market Scout уважает вашу конфиденциальность. Данный документ описывает, как мы обрабатываем данные при использовании нашего сервиса и API сторонних платформ.

1. Сбор информации

Мы собираем только общедоступную агрегированную информацию о трендах. Мы не собираем и не храним персональные данные пользователей Pinterest, их пароли или личные сообщения.

2. Использование API

Market Scout использует официальный Pinterest API для получения данных о популярных темах и категориях. Эти данные используются исключительно для формирования аналитических отчетов для авторов контента.

3. Защита данных

Все запросы к API передаются по зашифрованным протоколам SSL/TLS. Мы не передаем полученную аналитическую информацию третьим лицам для рекламных целей.

4. Права пользователя

Поскольку мы не храним персональные данные, у нас нет профилей пользователей, которые можно было бы изменить или удалить. Однако вы всегда можете связаться с нами по вопросам работы приложения.
